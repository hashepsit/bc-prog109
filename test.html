Module Assignments


Module 1: Module Webpage
Create a page that shows the main image of a recipe and five small images of other recipes. When clicking on each image, each image should take you to the recipe of that image.

Module 2: Tables
Utilize tables to present some information

Module 3: Cascading Style Sheets
Use an external CSS to style content on a .HTML file. Use external fonts, change heading styles, colors, and learn how to validate the CSS code.

Module 4: HTML Structure and CSS Styling
Use an external CSS to style content on a .HTML file. Use external fonts, change heading styles, colors, and learn how to validate the CSS code as well. Use 'div' to create different blocks and will learn how to change specific parts of an HTML section. Use lists to create a horizontal menu.

Module 5: JavaScript Multiplication Table
reate the first Javascript program by asking a user for some values and returning results to the HTML page. Apply the concepts of variables, loops and prompting the user for a value. Modify an existing example from Chapter 4and create a Multiplication table.

Module 7: TravelWorthy Hotel Availability - Objects
This module covered objects and native objects from Javascript. This assignment modifies existing code and add a new object to the page.

Module 8: ListKing - DOM Events
Exercise knowledge of DOM and some events to add an element to an existing list.

Module 9: Event Propagation and DOM Modification
Use a baseline code to learn about even propagation and how to change the behavior of a default event.
Create an HTML form that will draw a dot on the position where the user clicked. You also should provide an option to clean the screen.

Artifacts


Artifact 1: Resume
Create a one page, resume style, using HTML. Perform a Document-conformance checking by using the Nu HTML checker.

Artifact 2: Chessboard
Create a chessboard to exercise and master CSS concepts. Learn to distinguish between IDs, classes and HTML elements. Understand how to link an HTML file with a CSS file and to design the style from scratch without the need of using tables.

Artifact 3: JavaScript Rhombus
Use JavaScript functions, program structures and control flow to implement a basic JavaScript program. Understand how JavaScript can be used to modify the web environment and document. Generate a rhombus based on user's parameters

Artifact 4: JavaScript Form and Validation
Demonstrate the use of Javascript and knowledge of Document Object Model to validate a form.

Artifact 5: JavaScript Slideshow
Demonstrate the use of Javascript events and objects to modify the DOM and create a slideshow.
